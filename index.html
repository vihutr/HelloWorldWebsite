CSS and HTML Basics

CSS Fundamental Structure

selector{
    property: value;
}

A selector (such as h1, h2, p, or even body) altered with properties (such as color or opacity) of some value (like blue or gray). Properties blocks are always separated by semicolons.

The CSS code applied to multiple classes on a single particular selector go in order of how the code is written.

Example: The following button has the classes buttonEx, roundA, and roundB associated with it. While both roundA and roundB have property border-radius, A has a value of 20px (only slightly rounded reactangular button) while B has a value of 50% (essentially a fully rounded button). In this case, roundA's code is written in the CSS style section first and then roundB comes after. A executes first but B overwrites this execution, creating a completely round button. Copy pasting roundB's code before roundA's will have the opposite occur.

.buttonEx{
    background-color: rgb(0, 128, 255);
    border: none;
    color: rgb(255, 255, 255);
    padding: 20px;
    font-family: monospace;
    text-align: center;
    text-decoration: none;
    display: inline-block;
    font-size: 16px;
    margin: 4px 2px;
}
.roundA{ border-radius: 20px; }
.roundB{ border-radius: 50%; }

Common HTML

<div>A divider. The following are semantic div tags used for readability:
<header> The header
<nav> Some form of navigation. Most basic form tends to be a ul with many li's for each link.
<section> A section
<footer> The footer

Using these tags one can organize a webpage into sections, or boxes, that make up a complete web page.

Using CSS Margin (adding space around box) and Padding (adding space within box) properties allows for altering how the page looks. One value applies to each side, two values has 1st val to top and bottom, 2nd value to left and right. 4 values is top, right, bottom, left.
